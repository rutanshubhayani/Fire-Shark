## Run Locally

Clone the project

```bash
  git clone https://github.com/.git
```

Go to the project directory

```bash
  cd Fire-Shark
```

Install dependencies

```bash
  npm install
```

Create an ENV file and provide environment variables

```bash
  .env
```

Start the server

```bash
  npm run start
```

## Tech Stack

**Server:** Node, Express, MongoDB
